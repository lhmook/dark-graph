# Dark (Season 1-2) Character Graph

넷플릭스 드라마 *Dark*를 시청하며 헷갈리는 인물 관계를 완벽히 꿰뚫어보기 위해 제작된 **인터랙티브 가계도 및 인물 관계망 어플리케이션**입니다.

시즌 1을 넘어 이제 **시즌 2 시점까지의 완전한 관계도를 지원**하며, 시즌 필터를 통해 후반부의 치명적인 스포일러를 사전에 차단하는 강력한 기능을 제공합니다!

---

## 프로젝트 개요

* Netflix 드라마 *Dark* 시즌별 인물 관계 그래프 시각화
* React + Cytoscape 기반의 브라우저 최적화 뷰잉
* 시간에 따라 나이를 먹고 변하는 동일 인물의 연대기적 초상화 제공
* 복잡한 혈연 관계와 가족간 연결을 겹침 없이 보기 좋게 정렬한 다이내믹 UI

---

## 주요 기능 및 최신 업데이트

* **스포일러 방지 필터 (Spoiler Control):** 시즌 1~3 콤보박스 설정에 따라 아직 극 중에서 밝혀지지 않은 인물과 충격적인 가족 관계가 철저히 대기 상태로 숨겨집니다.
* **다중 세계(World) 및 연도 필터:** 아담의 세계(A), 이브의 세계(B), 근원 세계(Origin) 등 다중 우주 분기와 연도별 생존/활동 인물 전용 탐색 지원.
  *(※ 극의 연출 흐름에 맞춰 시즌 2까지는 World 기능을 숨기고 미스터리 요소를 가중시킵니다.)*
* **고정된 직관적 레이아웃 UX:** 물리 엔진 기반의 무작위 배치를 지양하고, 4대 가문(니엘센, 칸발트, 티데만, 도플러)을 좌우로 세우고 세대별로 높낮이를 맞추어 언제나 동일한 가계도 형태를 유지합니다.
* **스마트한 선 연결 (Edge Routing):** 가족 간 부모-자식 선은 교차를 피하는 회로망 형태(`taxi` 스타일 직각)로 뻗어 나가며, 연인이나 소속 관계 등은 포물선 형태(`bezier`)로 우회하여 노드의 얼굴을 가리지 않고 깔끔히 뻗어나갑니다.
* **사이드바 하단 범례표 & 상세 정보창:** 좌측 사이드바를 통해 복잡한 선들의 의미(부모, 부부, 연인, 남매, 동일인물)를 한눈에 파악할 수 있으며, 인물을 클릭해 시간에 따른 여러 얼굴을 확인할 수 있습니다.
